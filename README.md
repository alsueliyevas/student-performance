# Student Performance Data Analysis Project

## Overview
This project analyzes student performance using a dataset that includes demographic and educational factors. The main objective is to understand how different variables influence academic performance in mathematics, reading, and writing.

## Dataset Description
The dataset contains the following features:

- Gender
- Parental level of education
- Lunch type (standard or free/reduced)
- Race/ethnicity
- Test preparation course completion status
- Scores in mathematics, reading, and writing
- Internet usage
- Parental expectations
- Sleep schedule
- Anxiety,burnout,depression and stress levels

## Objective
The goal of this project is to analyze how demographic and educational factors affect student performance across three subjects: mathematics, reading, and writing. The focus is on identifying meaningful patterns and correlations in the data.

## Key Questions
- Does gender influence academic performance?
- How does parental education level affect student scores?
- What is the impact of test preparation courses on results?
- Does lunch type indicate differences in performance?
- Are there significant differences across demographic groups?

## Outcome
The analysis identifies the most influential factors affecting student performance and highlights patterns across different student groups and subjects.
